# 社区与交流

欢迎加入 PgFlow 社区，获取帮助、分享反馈和讨论想法。

## 反馈渠道

| 渠道 | 用途 |
|------|------|
| [GitHub Issues](https://github.com/leoyangx/PgFlow/issues) | Bug 报告、功能建议 |
| [GitHub Discussions](https://github.com/leoyangx/PgFlow/discussions) | 使用问题、经验分享、想法讨论 |

## 常见问题快速跳转

- **如何配置 Telegram Bot** → 管理面板 → 文档 → Telegram
- **如何切换 AI 模型** → 管理面板 → 配置 → AI 模型
- **日志在哪里看** → 管理面板 → 日志，或右键托盘图标 → 查看日志
- **数据存在哪里** → `~/.pgflow/`，永远不离开你的设备
